에라토스테네스의 체
핵심 아이디어 : 어떤 수가 소수가 아니라는게 밝혀지면, 그 배수들은 전부 소수가 아님 !

1.0과 1은 소수가 아니니까 미리 표시해둔다.
(보통 is_prime[0] = false, is_prime[1] = false)

2. 2부터 √N 까지만 확인하면 된다.
→ 왜냐? √N보다 큰 수로 나누어 떨어지면, 그보다 작은 수도 이미 나눠떨어졌을 것이기 때문

3.for (int i = 2; i * i <= MAX; i++)
{
if (아직 지워지지 않은 수라면) → 이 수가 소수다!
→ 그러면 이 수의 배수들을 전부 지운다 (소수가 아님 표시)
}

4.배수 지울 때 시작점은 i×i 부터 하면 더 효율적이다
(i×2, i×3, …, i×(i-1) 까지는 이미 앞에서 지워졌음)

5.지울 때는 보통 2칸씩 증가시키면서 지운다 (짝수는 이미 다 지워짐)
→ i가 2일 때는 2칸씩, i가 3 이상이면 i칸씩 증가

6.위 과정을 다 마치고 나면
is_prime[] 배열에서 true로 남아있는 인덱스들이 모두 소수!

7.마지막으로 M부터 N까지 돌면서
is_prime[i]가 true면 출력

1. 1000001 크기 bool 배열 준비 (false = 소수 아님)
2. 0, 1은 false로
3. 2 ~ √1,000,000 까지 반복
   └─ 소수라면 → 그 수의 제곱부터 배수들을 false로
4. M ~ N 까지 true인 것만 출력
