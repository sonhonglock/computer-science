이 문제의 진짜 깨달음 포인트는 다음 세 문장으로 압축된다.

각 창문 k는 자기 자신을 약수로 가지는 횟수만큼 토글된다.
약수의 개수가 홀수인 숫자는 완전제곱수뿐이다.
따라서 열려 있는 창문 = N 이하의 완전제곱수의 개수

결국 답은
⌊√N⌋ (N 이하의 가장 큰 정수의 제곱근)

근데 123까진 이해했는데
왜 갑자기 답이 N 이하의 가장 큰 정수의 제곱근임
N 이하에 존재하는 완전제곱수는 정확히 아래와 같습니다:
1², 2², 3², … , k²    (단, k² ≤ N)
→ 가장 큰 k는? → k ≤ √N
→ 정수이므로 k = ⌊√N⌋ (가장 큰 정수 k)
→ 그러니까 1부터 ⌊√N⌋까지의 정수 각각이 제곱수를 하나씩 만들어냄
→ 개수 = ⌊√N⌋ 개

#include <stdio.h>
#include <stdbool.h>
#include <math.h>
#include <string.h>

int main() {
    long long N;
    scanf("%ld",&N);
    printf("%ld",(long long)sqrt(N));
}
